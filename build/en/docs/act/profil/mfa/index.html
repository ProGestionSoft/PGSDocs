Multi-Facteur Authentification (MFA)

Accéder rapidement à « SECURITE »

Pour accéder rapidement à la section DOCUMENTS, saisissez simplement dans la barre de recherche un mot-clé de recherche pour trouver la section sans naviguer manuellement dans les menus.

Voici une liste – non exhaustive – des mots clés de recherches pris en charge.

La sécurité des comptes est une priorité pour PRO GESTION SOFT. Pour mieux protéger vos informations personnelles et sensibles, nous vous offrons la possibilité d’activer une authentification multi-facteur (2FA). Ce système ajoute une couche de sécurité supplémentaire en demandant une validation lors de chaque connexion, afin de confirmer que vous êtes bien l’utilisateur légitime de votre compte.

Qu'est-ce que l'Authentification Multi-Facteur ?​

L'authentification multi-facteur, souvent abrégée en 2FA ou MFA, est une méthode de sécurité qui exige deux étapes de vérification pour confirmer votre identité. En plus du mot de passe habituel, elle ajoute une vérification via un autre moyen, ici par e-mail, pour garantir que seul vous pouvez accéder à votre compte.

L'authentification multi-facteur est cruciale pour :

  • Prévenir les accès non autorisés en ajoutant une vérification supplémentaire.
  • Renforcer la sécurité des données sensibles et protéger votre compte contre les tentatives de piratage.
  • Donner plus de sérénité aux utilisateurs en ajoutant une barrière de sécurité.

Activer l'Authentification Multi-Facteur​

Suivez les étapes ci-dessous pour activer la double authentification sur votre compte :

  • Connectez-vous à votre compte.
  • Dans l'interface de navigation, cliquez sur votre icône de profil pour ouvrir le menu.
  • Cliquez sur Développer pour accéder à votre espace profil.
  • Dans votre espace profil, accédez à la section Sécurité grâce au onglets à droit.
  • Sous la section Two-Factor Authentication, cliquez sur le bouton Activer.
  • Une fois cette étape effectuée, la double authentification par e-mail sera activée sur votre compte.

Comment fonctionne l'authentification par e-mail ?​

Désormais, chaque fois que vous vous connecterez, vous devrez passer par une validation en deux étapes :

  • Connexion avec vos identifiants : Saisissez votre adresse e-mail et votre mot de passe habituels pour vous connecter.
  • Vérification par e-mail : Un e-mail de vérification contenant un code de validation sera envoyé à l'adresse e-mail associée à votre compte. Ce code est à usage unique et expire en 15 minutes.
  • Saisir le code : Entrez le code reçu dans l’interface de connexion pour finaliser l’accès à votre compte.
DUREE DE VALIDITE DU CODE

Le code de validation envoyé par e-mail est temporaire. Assurez-vous de l’utiliser rapidement, car il expirera après 15 minutes pour des raisons de sécurité. Si le code expire, vous devrez recommencer le processus en demandant un nouveau code.

Que faire en cas de perte d'accès à l'e-mail ?​

Dans le cas où vous ne pouvez plus accéder à l'adresse e-mail associée à votre compte, suivez les étapes suivantes pour éviter tout blocage :

  • Mise à Jour de l’adresse si connecter : Si vous êtes toujours connectés, mettez à jour rapidement l’adresse e-mail avant d’activer la double authentification. Cela garantit que le code de vérification sera envoyé à une adresse fonctionnelle.
  • Contactez le support technique : En cas de perte d’accès définitive à l’e-mail, contactez l’équipe de support qui pourra vous aider à rétablir votre accès après vérification de votre identité.
Recommandations pour une utilisation optimale du MFA
  • Gardez votre e-mail sécurisé : Assurez-vous que votre adresse e-mail est sécurisée, avec un mot de passe fort et une authentification 2FA si possible. Votre compte PRO GESTION SOFT dépend de la sécurité de cet e-mail.
  • Ne partagez pas votre code : Le code de vérification reçu par e-mail est personnel. Ne le partagez avec personne pour garantir la confidentialité de votre compte.
  • Surveillez les connexions suspectes : Si vous recevez un code de vérification sans avoir tenté de vous connecter, cela pourrait indiquer une tentative d’accès non autorisée. Dans ce cas, changez immédiatement votre mot de passe et contactez le support si nécessaire.

Désactiver la double authentification​

Si vous souhaitez désactiver la double authentification par e-mail, vous pouvez le faire depuis la section Sécurité de votre Espace Profil. Suivez les mêmes instructions au niveau de ‘’Activer l'Authentification Multi-Facteur‘’. Le bouton sera désormais intitulé « Désactiver ».

warning

Notez que désactiver la double authentification réduira le niveau de sécurité de votre compte. Il est recommandé de la garder active pour protéger vos informations.
