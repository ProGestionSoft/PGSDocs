Playground

Cet environnement a été conçu pour vous permettre de découvrir, tester et expérimenter les fonctionnalités de notre plateforme dans un cadre convivial et pratique.

Accédez au Playground et sélectionnez un simulateur spécifique du module que vous souhaitez explorer.

Ce que vous pouvez faire​

Vous avez accès à des outils interactifs pour :

  1. Éditeur interactif : Écrivez et modifiez du code en direct. Les résultats s'afficheront en temps réel, vous permettant de tester vos idées immédiatement.

  2. Aperçu uniquement : Si vous préférez une expérience plus légère, vous pouvez simplement voir un aperçu des fonctionnalités sans plonger dans le code.

  3. Code uniquement : Observez le code source de chaque simulateur. Ceci peut être le point de départ pour vous permettre de démarrer rapidement vos projets.

warning

Nous appliquons par défaut des formatages avancés aux simulateurs "Aperçu". Le simulateur de rendu interactif prend en charge d'autres formatages. Il vous reviendra de créer et d'intégrer vous-même vos styles personnalisés, en fonction de l'expérience que vous souhaitez obtenir.

Précautions et manipulation du code​

Afin de garantir une expérience optimale et sécurisée, nous vous demandons de respecter les consignes suivantes :

  1. Utilisation responsable : Le Playground est un environnement d’apprentissage. Évitez d’exécuter du code malveillant ou de soumettre des données sensibles.

  2. Données simulées : Toutes les données utilisées sont fictives. Ne tentez pas de tester des informations réelles ou confidentielles dans cet espace.

  3. Sauvegarde du code : Assurez-vous de sauvegarder vos codes et configurations localement avant de quitter le Playground. Les modifications effectuées ici ne sont pas enregistrées de manière permanente.

  4. Compatibilité des navigateurs : Pour une expérience fluide, nous recommandons l’utilisation d’un navigateur moderne et à jour.

En explorant ce Playground, vous développerez vos compétences, testerez vos idées et contribuerez à enrichir notre communauté. Amusez-vous et apprenez en toute sérénité !
