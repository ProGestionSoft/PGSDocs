# <i class="fas fa-clipboard-question"></i> A propos des tests en ligne
<img src="/img/docs/test.png" alt="TEST EN LIGNE" className={styles.docsImg} />

Les tests en ligne sont parfois requis dans le cadre des offres de recrutement. Ces tests permettent aux recruteurs d’évaluer les compétences des candidats avant de prendre une décision. Voici comment vous pouvez participer à ces tests.

## Notification de test en ligne \{#notification}
Lorsque vous postulez à une offre et qu’un test en ligne est exigé, vous recevrez un mail de notification après la soumission de votre candidature. Ce mail vous indiquera clairement si un test doit être passé.

:::warning
Certaines offres qui initialement ne mentionnaient pas de test en ligne peuvent, à la discrétion du recruteur, en ajouter un plus tard. Pour cette raison, il est essentiel de ***suivre régulièrement les informations relatives à votre offre*** ou ***suivre le recruteur*** afin d’être informé immédiatement de tout ajout ou modification concernant le processus de recrutement.
:::


## Passer un test en ligne \{#test}
Voici les étapes pour passer votre test en ligne une fois que vous avez été informé de son existence :

1. **Accéder au test via le mail reçu**  
  Après la soumission de votre candidature, vous recevrez un mail vous invitant à passer un test. Ce mail contiendra un lien vers le test, accompagné d’un bouton ***"Passer les tests maintenant"***. En cliquant sur ce bouton, vous serez redirigé vers la plateforme de tests en ligne, où vous pourrez commencer à répondre aux questions.

2. **Accéder aux détails de l'offre**  
  Si vous n'avez pas reçu de mail ou si vous souhaitez vérifier les informations relatives à l’offre, vous pouvez également vous rendre directement sur la page des détails de l'offre. Sur cette page, vous devrez renseigner votre adresse email pour être redirigé vers la page des tests en ligne.

3. **Répondre aux questions**  
  Les tests en ligne sont généralement composés de questions à choix multiple (QCM) et sont conçus pour évaluer vos compétences dans des domaines spécifiques liés à l'offre d'emploi. Prenez votre temps pour répondre à chaque question de manière réfléchie. Il est également possible que le test comporte plusieurs sections, chacune testant une compétence différente.

4. **Soumettre votre test**  
  Une fois toutes les questions répondues, vous aurez la possibilité de soumettre votre test. Assurez-vous d’avoir répondu à toutes les questions avant de soumettre, car après la soumission, il ne sera plus possible de modifier vos réponses.

:::tip ***Préparation et conseils***
- Avant de commencer le test, assurez-vous de ***bien comprendre les consignes***.
- Assurez-vous d'avoir un environnement calme et un matériel approprié avant de commencer, afin de ***répondre aux questions sans distractions***.
- Bien que les tests en ligne soient importants, ils ne définissent pas (peut-être) tout. ***Prenez votre temps pour réfléchir*** à chaque question.
- Si un test est ajouté après la soumission de votre candidature, assurez-vous de ***vérifier régulièrement les informations*** relatives à l'offre ou suivez le recruteur.
:::
