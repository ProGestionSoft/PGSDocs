# <i class="fas fa-code"></i> Introduction  

Bienvenue dans l’espace développeurs de **Pro Gestion Soft** ! Cette documentation a été spécialement conçue pour vous accompagner dans la personnalisation et l’intégration de nos outils dans vos projets professionnels. Que vous soyez un intégrateur, un analyste ou un créateur de solutions, vous trouverez ici les ressources nécessaires pour exploiter pleinement le potentiel de notre plateforme.  

Dans cette section, nous allons explorer des fonctionnalités clés, organisées pour répondre aux besoins variés des développeurs :  

1. **Intégration d’iFrames**  
   Découvrez comment extraire des widgets interactifs et dynamiques depuis la bibliothèque cloud de PGS et les intégrer facilement dans des sites web tiers. Grâce à ces iFrames, vous pouvez enrichir vos interfaces utilisateur avec des fonctionnalités prêtes à l’emploi tout en centralisant vos données.  

2. **Création de sites statistiques**  
   Nous vous montrons comment concevoir et intégrer des sites statiques directement dans la plateforme PGS. Cette section détaille les processus pour créer des visualisations de données dynamiques, essentielles pour prendre des décisions stratégiques basées sur des indicateurs personnalisés. 

3. **Référence API**  
   Plongez dans notre documentation API pour comprendre comment interagir avec les différents modules de PGS (gestion des utilisateurs, facturation, bibliothèque cloud, etc.). Vous apprendrez à effectuer des appels API sécurisés, manipuler des données en temps réel et automatiser vos flux de travail.  

4. **Playground**  
   Profitez d’un espace interactif dédié pour tester et expérimenter en temps réel. Le playground vous permet de manipuler du code, d'explorer des exemples pratiques et de valider vos intégrations avant de les implémenter.
