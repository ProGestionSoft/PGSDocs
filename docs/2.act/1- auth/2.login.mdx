# <i className="fas fa-sign-in-alt"></i> Connexion à votre compte
<img src="/img/docs/login.png" alt="CONNEXION" className={styles.docsImg} />

La connexion à votre compte sur **PRO GESTION SOFT** est conçue pour être rapide, sécurisée et simple. Si vous êtes déjà inscrit, vous pouvez facilement vous connecter à l’adresse `https://progestionsoft.com/login`, en utilisant votre adresse e-mail et votre mot de passe.

Voici un guide détaillé pour vous accompagner dans cette étape, ainsi que les options disponibles en cas de perte de mot de passe.


## Étape de connexion \{#step}
Pour accéder à votre compte, suivez les étapes ci-dessous :

-	Saisissez votre ***adresse e-mail*** : Veillez à entrer l’adresse e-mail que vous avez utilisée lors de votre inscription. Assurez-vous de la saisir correctement, car même une petite erreur peut empêcher la connexion.
-	Entrez votre ***mot de passe*** : Le mot de passe doit être identique à celui que vous avez défini lors de votre inscription. Pour des raisons de sécurité, votre mot de passe est masqué par des symboles lors de la saisie afin de protéger vos informations contre toute tentative de regard indiscret.
-	Cliquez sur le bouton <span className={styles.blueBtn}>Connexion</span> : Une fois vos identifiants saisis, cliquez sur le bouton connexion pour soumettre vos informations. Nous vérifierons que les informations correspondent à un compte existant. Si elles sont correctes, vous serez redirigé vers la page d’accueil de votre compte.


## En cas d'oubli du mot de passe \{#forgot-password}
Nous comprenons qu’il est possible d’oublier son mot de passe. Pour faciliter la récupération d'accès à votre compte, nous avons mis en place une procédure de réinitialisation simple et sécurisée. Voici comment procéder :

-	Cliquez sur ***"Mot de passe oublié"*** sur la page de connexion, juste en dessous du bouton "Connexion" afin de démarrer le processus de réinitialisation.
-	Renseignez votre ***adresse e-mail*** : Une nouvelle page s’ouvrira, vous invitant à entrer l’adresse e-mail associée à votre compte. Cette étape est cruciale, car elle nous permet de vérifier votre identité en envoyant un code de six (6) chiffres à une adresse vérifiée.
-	Réception du ***code de vérification*** : Une fois votre adresse e-mail soumis, nous vous enverrons un e-mail contenant un code à six (6) chiffres pour réinitialiser votre mot de passe. Ce code est valable pendant 15 minutes.
-	Création d'un ***nouveau mot de passe*** : En entrant dans le champ de saisie, le code envoyé par mail, vous serez dirigé vers une page où vous pourrez choisir un nouveau mot de passe. Assurez-vous de créer un mot de passe solide, en suivant les recommandations de sécurité (minimum de huit caractères, avec des lettres majuscules et minuscules, des chiffres et des symboles).
-	***Connexion*** avec le nouveau mot de passe : Une fois votre mot de passe réinitialisé, vous pourrez retourner à la page de connexion et utiliser votre nouvelle combinaison d'adresse e-mail et de mot de passe pour accéder à votre compte.

> Si vous perdez accès à votre adresse mail, contactez l’équipe de support technique qui vous guidera dans la récupération de votre compte après vérification de votre identité.

:::tip ***Conseils pour assurer la sécurité de votre compte***
Pour maintenir un niveau de sécurité élevé pour votre compte, voici quelques bonnes pratiques à adopter :
-	***Changez régulièrement votre mot de passe*** : La mise à jour régulière de votre mot de passe permet de limiter les risques de compromission de votre compte.
-	***Ne partagez pas vos informations d'identification*** : Gardez votre mot de passe privé et évitez de le partager avec d'autres personnes, même celles en qui vous avez confiance.
-	***Utilisez un gestionnaire de mots de passe*** : Si vous avez du mal à mémoriser un mot de passe complexe, envisagez d'utiliser un gestionnaire de mots de passe sécurisé pour stocker et protéger vos informations.
-	***Surveillez les connexions suspectes*** : Si vous remarquez des activités inhabituelles sur votre compte, telles que des connexions que vous n’avez pas effectuées, contactez immédiatement notre équipe d'assistance pour prendre les mesures nécessaires.
:::
