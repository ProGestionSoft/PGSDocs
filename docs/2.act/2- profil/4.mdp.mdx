Votre mot de passe est un élément essentiel de la sécurité de votre compte sur **PRO GESTION SOFT**. Il est important de savoir comment le gérer, que ce soit pour le modifier, le réinitialiser ou simplement adopter de bonnes pratiques pour renforcer la protection de vos données.

## Pourquoi changer ou réinitialiser son mot de passe ? \{#why}
Il existe plusieurs raisons pour lesquelles vous pourriez vouloir modifier ou réinitialiser votre mot de passe :

- ***Sécurité renforcée*** : changer régulièrement votre mot de passe réduit les risques d'accès non autorisé.
- ***Suspicion d'accès non autorisé*** : si vous pensez que votre mot de passe a été compromis, il est impératif de le changer immédiatement.
- ***Amélioration de la robustesse*** : Si votre mot de passe actuel ne répond pas aux critères de sécurité recommandés (longueur, complexité, etc.), il est conseillé de le mettre à jour.


## Comment modifier ou péinitialiser son mot de passe ? \{#how}
PRO GESTION SOFT offre deux moyens simples pour gérer votre mot de passe : la ***réinitialisation*** et la ***modification*** directement.

### Réinitialisation via l'option « Mot de passe oublié » \{#reset}
Si vous ne vous souvenez plus de votre mot de passe ou si vous n’arrivez pas à vous connecter, suivez ces étapes, accédez à la page de connexion, et suivez les instructions [‘’d'oubli du mot de passe‘’](/docs/act/auth/login#forgot-password)

> Le lien de réinitialisation est valable pendant une durée limitée pour des raisons de sécurité. Si le délai expire, vous devrez recommencer le processus.

### Modification du mot de passe via l’Espace Profil \{#edit}
Si vous souhaitez changer votre mot de passe alors que vous êtes déjà connecté, procédez ainsi :

1. Connectez-vous à votre compte et cliquez sur votre icône de profil dans le coin supérieur droit pour accéder à votre l’***Espace Profil***.
2. Dans votre Espace Profil, Naviguez vers la ***section Sécurité***, puis trouvez la sous-section mot de passe.
3. Saisissez votre ***mot de passe actuel*** afin de vérifier que vous êtes bien le propriétaire du compte.
4. Entrez un ***nouveau mot de passe*** répondant aux critères de sécurité requis (voir ci-dessous) et confirmez-le en le tapant à nouveau dans le champ de confirmation pour éviter les erreurs de saisie.
5. Validez le changement en cliquant sur le bouton <span className={styles.blueBtn}>Enregistrer</span>.

:::tip Bonnes Pratiques pour un Mot de Passe Sécurisé
Pour garantir la sécurité de votre compte, voici quelques recommandations :
- Utilisez un ***mot de passe Long et complexe***,
- Minimum de ***huit (8) caractères***,
- Inclure une combinaison de ***lettres majuscules, lettres minuscules, chiffres et caractères spéciaux (@, #, $, %)***.
- ***Évitez les mots simples*** comme "123456" ou "password".
- ***Évitez de réutiliser vos mots de passe***,
- N’enregistrez pas vos mots de passe dans des ***emplacements non sécurisés***,
- ***Ne partagez jamais votre mot de passe*** ; même à une personne prétendant être un représentant de PRO GESTION SOFT. A la place, accordez un accès co-administrateur.
- Méfiez-vous des ***e-mails ou messages suspects demandant vos informations de connexion***.Nous ne vous demanderons jamais votre mot de passe par e-mail.
:::
