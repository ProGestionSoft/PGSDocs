# <i class="fas fa-palette"></i> Préférences
<img src="/img/docs/pref.png" alt="PREFERENCES" className={styles.docsImg} />

Les préférences vous permettent de personnaliser votre expérience sur en ajustant des paramètres tels que la langue et le thème. Ces options vous permettent de travailler dans un environnement qui vous est le plus confortable, en fonction de vos besoins et préférences personnelles.


## Thème \{#theme}
Le thème visuel de l'interface joue un rôle important dans le confort d'utilisation. Nous proposons deux (02) modes d'affichage principaux :
- ***Thème Clair*** : Ce thème est le mode par défaut et offre une interface lumineuse et claire, idéale pour une utilisation en journée ou dans des environnements lumineux.
- ***Thème Sombre*** : Le mode sombre est conçu pour une expérience visuelle plus douce, particulièrement utile pour les environnements peu éclairés ou pour les utilisateurs préférant un fond sombre.

:::warning Limitations du Thème Sombre
Bien que le thème sombre soit pris en charge, il peut contenir des incohérences ou des éléments visuels qui ne sont pas entièrement optimisés. Si vous rencontrez des problèmes avec ce mode, nous vous encourageons à notifier le support pour signaler toute incohérence ou élément manquant.
:::


## Langue \{#langue}
Nous supportons deux (02) langues et vous pouvez choisir celle qui vous convient. Actuellement, les deux langues principales disponibles sont le français (par défaut) et l'anglais.


:::note Mises à jour et traductions
Nous nous s'efforcons de maintenir des traductions précises et à jour. Toutefois, certaines traductions peuvent ne pas être entièrement cohérentes en raison des mises à jour fréquentes.

Si vous détectez une mauvaise traduction ou une incohérence dans le texte, vous avez la possibilité de notifier le support, soumettre une nouvelle traduction directement via le bouton "***Proposer une traduction*** présent dans cette documentation, ou en accédant au **[dépôt GitHub de traduction](https://github.com/ProGestionSoft/PGSTranslate)**.
:::


## Configuration \{#config}
Pour personnaliser vos préférences, 
- Accédez à la Barre de Contrôle en haut de l'interface.
- Cliquez sur l'icône <i class="fas fa-cog"></i> "Préférence" pour ouvrir les options de personnalisation.
- Selectionnez l'option de personnalisation (Thème ou Langue) que vous souhaiter modifier.
- Choisir la valeur que vous souhaiter utiliser.
